ARLE

About The Project

Automated Robot for Library Enhancement is an effort to make an automated mobile robot capable of managing books in the institute library (picking and placing them in the correct racks) using an arm placed on a mobile robot. It is a very cumbersome task to locate a particular book and put it back on the same rack. There is also a big scope of human errors if done by humans and it becomes difficult to correct these errors. ARLE is primarily being developed to manage libraries but its use can be extended to warehouses, shops, inventories and restaurants.
